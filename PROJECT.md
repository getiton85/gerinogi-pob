# 게리노기 지멋대로 POB 밸류

마비노기 모바일 검술사 세팅을 룬, 패시브, 스탯, 조건부 효과 기준으로 비교하는 정적 웹앱입니다.

## 현재 원칙

- 기존 기능 유지 후 새 기능 추가
- 룬 밸류는 현재 장착 세팅 대비 최종 기대딜 변화율 기준
- 속도, 쿨타임, 회복 계열은 룬 추천 점수에 직접 가산하지 않고 스킬별 쿨/1분 피해에서 분리 표시
- UI에는 유틸 옵션을 표시할 수 있지만 추천 순위는 기대딜 기준으로 정렬
- 임시 데이터와 의미 없는 이름은 배포본에 남기지 않음

## 현재 주요 파일

- index.html: 앱 진입점
- style.css: 화면 배치와 반응형 스타일
- data.js: 룬, 직업, 패시브 DB
- core.js: 스탯 환산, 룬 효과 합산, 기대딜 밸류 계산, 자체 검수
- app.js: 장착 UI, 룬 선택창, 대시보드, 추천 순위 렌더링

## GitHub Pages 배포

이 앱은 별도 서버 없이 브라우저에서 실행되는 정적 웹앱입니다. 저장소 루트에 현재 파일들을 올리고 GitHub Pages를 저장소 루트 또는 배포 브랜치로 지정하면 웹에서 볼 수 있습니다.

# 버전 표시 규칙

- 배포되는 기능, 데이터, 계산식, UI가 바뀔 때마다 좌측 상단 버전 숫자를 반드시 올린다.
- 기준 파일은 `version.json`이며, 화면 표시는 `index.html`의 `#appVersion`이다.
- `style.css`, `data.js`, `core.js`, `app.js` 참조에는 같은 버전 쿼리를 붙여 GitHub Pages/브라우저 캐시를 줄인다.
- Codex는 수정 작업을 완료하기 전에 버전 증가 여부를 검수한다.
