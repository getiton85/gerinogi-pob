# 마비노기 모바일 POB v12 Trusted

UI는 v11 형태를 유지하면서 내부 계산엔진을 core.js로 분리했습니다.

검수 포함:
- DB 중복 ID 검사
- 엠블럼/무기/방어구 개수 검사
- 기본 장착 ID 검사
- 평균 점수 산출 검사
- 패시브 ON/OFF 반응 검사
- 레이드/어비스 모드 반응 검사
- 직접DPS 산출 검사

주의:
게임 내부 공식이 공개되지 않은 부분은 추정 모델입니다.


v12 QA: Node 문법 검사 및 자체 검수(runSelfTest) 통과.

v13 수정:
- 스탯창 입력: 공격력/방어력만 상단 분리
- 두 번째 스탯창 입력: 브레이크, 강타 강화, 콤보 강화, 스킬 위력, 광역 강화, 회복력, 급소 회피, 추가타, 피해 감소, 빠른 공격, 연타 강화, 빠른 스킬, 추가 체력, 궁극기, 치명타 입력 추가
- 기타 자동 환산: 치명확률/추가타/강타피해/연타피해 자동 계산
- 계산 엔진은 수동 기본값 대신 두 번째 스탯창 기반 환산값을 사용

v14 수정:
- 두 번째 스탯창의 모든 주요 수치가 밸류점수에 반영되도록 통합 엔진 보강
- 치명/추가타/강타/연타뿐 아니라 브레이크, 스킬 위력, 광역 강화, 빠른 공격, 빠른 스킬, 피해 감소, 회복력, 급소 회피, 추가 체력, 궁극기까지 보정값에 포함
- 대시보드에 스킬계수/속도보정/브레이크/피해감소 추가 표시
- 밸류점수 공식 설명 UI 추가
- 자체 테스트에 '두 번째 스탯창 대시보드 반영' 항목 추가


v16 수정: 밸류점수를 기준 세팅 100.00으로 정규화. 버전이 올라가도 점수가 폭증하지 않도록 수정.


v18 수정: 기본 select 대신 커스텀 룬 선택창으로 변경. 상승분은 빨간색, 하락분은 파란색으로 표시. v16 안정본 기반으로 재적용.

v19 수정:
- 공격력, 치명타 확률, 추가타 확률, 치명타 피해를 고밸류 스탯으로 반영
- 밸류 가중치: 공격력 1.08 / 치명타 확률 1.18 / 추가타 확률 1.15 / 치명타 피해 1.12
- 정규화 방식은 유지: 현재 기준 세팅 100.00


v20 수정: 상단 제목을 '게리노기 지멋대로 POB밸류'로 변경, QA 문구를 버전 표기로 변경, 왼쪽 장착 패널 독립 세로 스크롤 적용.


v22 수정: v20 안정본 기준. 입력 스피너 제거, 자동환산 문구 제거, 브레이크~피해감소 추가분 표시, 보조스탯 100% 기준 환산 보정, 상위 레이아웃 불변.


v23 수정: 브레이크/스킬위력/광역강화/빠른공격/빠른스킬/피해감소가 10배로 표시되던 퍼센트 단위 오류 수정. 내부 계산도 추가분 기준으로 통일.


v24 1단계 구현: 버전 표기 수정, 기본 스탯값 정상화, 보조스탯 자동환산 10배 표시 오류 수정 구조 적용.


v25 임시본: 룬 DB 전수검사 추가, 무기룬 임시 표기 '저주의 룬 A/B'를 '저주의 룬: 억압/파멸'로 수정.


v26 검수판: 룬 밸류를 단순 합산점수에서 현재 기준 대비 최종 기대딜 변화율로 변경. 공격력/피해축/치명/추가타/유틸/직접DPS/생존축을 곱연산 기반으로 반영.
