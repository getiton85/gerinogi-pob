# 개발 로드맵

## 1단계: 현재 기능 안정화

- 룬 DB 무결성 유지
- 기대딜 밸류 계산 검수
- 장비 슬롯별 룬 선택 정렬 안정화
- GitHub Pages 배포 가능한 정적 구조 유지

## 2단계: 스킬 DB

- 검술사 스킬부터 DB화
- 스킬별 계수, 태그, 조건부 효과 추가
- 강타, 연타, 보조, 궁극기, 채널링 태그 반영

## 3단계: 장신구 시스템

- 장신구 기본 스탯
- 특정 스킬 강화
- 특정 스킬 변형
- 스킬 계산기와 연결

## 4단계: 직업별 계산기

- 직업 DB 분리
- 직업별 스킬 사이클
- 직업별 유효 옵션 규칙

## 5단계: 배포 품질

- 기준 숫자 고정
- 역검증 케이스 확장
- UI 검수 항목 문서화
- 회귀 검수 자동화
